`Ctrl+R` 反向搜索历史命令

`Ctrl+A` 光标移动到行的开头
`Ctrl+E` 光标移动到行的结尾
`Ctrl+B` 光标向后移动一个位置
`Ctrl+F` 光标向前移动一个位置
`Esc+B` 光标向左移动一个字符
`Esc+F` 光标向右移动一个单词

`Ctrl+U` 剪切从行的开头到光标前一个位置的所有字符
`Ctrl+K` 剪切从光标位置到行末的所有字符
`Ctrl+D` 删除光标位置的一个字符
`Ctrl+H` 删除光标位置的前一个字符
`Ctrl+W` 删除光标位置的前一个单词
`Ctrl+Y` 粘贴 ctrl+u或者 ctrl+k 剪切的内容
`Ctrl+&` 恢复 ctrl+h或者 ctrl+d 或者 ctrl+w 删除的内容
